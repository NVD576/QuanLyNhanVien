# 📌 Ứng dụng Quản Lý Nhân Viên

Ứng dụng desktop đơn giản cho phép quản lý nhân viên với các chức năng cơ bản:

## 🚀 Chức năng
- 🔑 Đăng nhập hệ thống  
- ➕ Thêm nhân viên  
- ❌ Xóa nhân viên  

## 👤 Tài khoản mặc định
- **Username:** `admin`  
- **Password:** `admin`  

## 🛠️ Công nghệ sử dụng
- C# WinForms (hoặc WPF, nếu bạn dùng WPF)
- .NET Framework / .NET (tùy project của bạn)
